# 🛡️ Security-Jobs

**セキュリティ求人一覧 | Security Jobs Japan**

日本国内のサイバーセキュリティ関連求人をまとめた一覧サイトです。
ベンダー・コンサルティングファーム・事業会社など、さまざまな業界のセキュリティポジションを掲載しています。

🔗 **公開URL**: https://naocyber.github.io/Security-Jobs/

---

## 📋 掲載内容

- 企業名（日本語・英語表記）
- 業界・企業概要
- ポジション概要・業務内容
- 年収レンジ
- 必要言語（日本語・英語レベル）
- セキュリティ領域タグ（ペネトレ/脆弱性診断、SOC/IR、GRC、クラウド、IAM など）
- 必要なツール・技術スタック

## 🔍 主な機能

- **フィルター機能**: 業界・セキュリティ領域・英語要件・リモート可否・優先度で絞り込み
- **検索機能**: 企業名・職種・キーワードでの検索
- **応募フォーム**: 各企業カードから直接応募が可能（フォーム送信は自動でメール通知）

## 🛠️ 技術構成

- 静的HTML / CSS / JavaScript（フレームワーク不使用）
- GitHub Pagesでホスティング
- フォーム送信は [FormSubmit](https://formsubmit.co/) を使用

## 📝 更新方法

1. `index.html` を編集（求人データは `JOBS` 配列に格納）
2. GitHubへコミット → 数分でPagesに反映
